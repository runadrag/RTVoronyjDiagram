    Voronyj (Voronoi) diagrams are being used for more than 300 years in epidemiology, architecture, mining and machine learning. Now we not only provide the support for it in Roassal 2, but also suggest an approach to visualize a software with voronyj diagrams.

    To download this just choose one method from the nexts:

    1. download RTVoronyjDiagram.zip from CI server;

    2. run Gofer new url: 'ttp://smalltalkhub.com/mc/NataliaTymchuk/RTVoronyjDiagram/main'; package: 'ConfigurationOfRTVoronyjDiagram'; load. (#ConfigurationOfRTVoronyjDiagram asClass project version: 'development') load.

    3. download throught Monticello using the repository: `MCHttpRepository

      location: 'http://smalltalkhub.com/mc/NataliaTymchuk/RTVoronyjDiagram/main'

      user: ''

      password: ''`
